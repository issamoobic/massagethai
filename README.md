# Мастерская тайских массажных практик — демо-лендинг

Интерактивный демо-прототип сайта авторской студии одного мастера в Новосибирске.
Визуальная концепция — **Modern Sanctuary** (кремовый + глубокий индиго + медь + коралл).

Назначение: показать клиенту (владельцу студии) визуал, UX онлайн-записи и работу
цифрового ассистента. Данные в JSON-файлах, заявки пишутся в `localStorage`.

## Запуск локально

```bash
cd massagethai
npm install
npm run dev
```

Откроется на `http://localhost:5173`.

## Стек

- **Vite + React 18 + TypeScript**
- **TailwindCSS 3** — дизайн-токены в `tailwind.config.ts`
- **Framer Motion** — scroll-reveal, переходы секций, плавающие декоративные элементы
- **react-hook-form + zod** — валидация формы записи
- **fuse.js** — нечёткий поиск по FAQ для чат-ассистента
- **lucide-react** — иконки
- Шрифты Google Fonts: Fraunces (дисплей) + Space Grotesk (UI)

## Что реализовано

- **Hero** — крупная типографика, парallax-декор, декоративный медальон с плавающими бейджами
- **О мастере** — цитата, биография, три принципа, техники и сертификаты
- **Программы** — тёмная секция с фильтром-чипами по категориям, модалка с подробностями, кнопка перехода к записи
- **Онлайн-запись** — 3 шага: программа → календарь + слоты → контакты. Слоты генерируются на 14 дней вперёд относительно текущей даты. Submit → `localStorage` + `console.info`
- **Отзывы** — слайдер с активным отзывом + сетка превью
- **FAQ** — аккордеон с анимированным открытием
- **Контакты** — Яндекс.Карта, мессенджеры, ссылка на ВК-сообщество
- **Чат-ассистент** — плавающий виджет справа внизу:
  - Quick-replies: "Подобрать программу", "Свободные слоты", "Цены и услуги", "Задать вопрос"
  - Квиз из 3 шагов с рекомендацией программы по весам
  - Показ 5 ближайших свободных слотов с переходом к форме
  - Fuzzy-поиск по FAQ (fuse.js)
  - История сообщений хранится в памяти компонента

## Как посмотреть сохранённые заявки

Откройте DevTools → Application → Local Storage → ключ `massagethai:bookings`.
Также каждая заявка логируется в консоли.

## Что заменить перед продом

- Тексты в `src/data/*.json` — особенно `master.json` (био, цитата, сертификаты), `reviews.json`, контактный телефон и адрес в `Contacts.tsx`, `Footer.tsx`.
- Плейсхолдер-портрет в `About.tsx` — на фото мастера (сохранить пропорции 4:5, рамка сохранится).
- Яндекс-карта в `Contacts.tsx` — подставить реальные координаты в `src=`.
